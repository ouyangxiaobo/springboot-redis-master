# springboot-redis-master
springboot整合redis做缓存处理
一、这只是一个简单的redis demo,
  1.安装redis数据库，修改配置文件中的相关参数
  2.导入数据库，修改配置文件中，数据库的账户密码，
  3.启动项目swagger 测试即可
  
二、
   demo采用的技术栈
   1.springboot2.1.6.RELEASE版本
   2.数据库为mysql
   3.采用mybatis自动生成代码工具
   4.swagger在线api管理
   5.redis可视化工具为FastoRedis
——————————————————————————————————————————
注意：idea需要安装lombok插件
